== Description ==
The Bulletin News theme is excellent for news, magazines, publishing, and review sites.Bulletin News is a blog and magazine-specialized WordPress theme. The theme is lightweight and performs at high speeds.Bulletin News is widget ready and comes with many blog/magazine-customized widget options. This theme makes website design an effortless task, for novice and expert users alike. Get polished results like the pros without writing a line of code.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
